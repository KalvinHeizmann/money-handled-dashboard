# Money Handled — Dashboard (v2)

The Money Handled dashboard UI: a single, self-contained HTML file. Editorial
"ledger" design, light + dark themes, no build step and nothing to install.

## Use it

- **Preview:** open `index.html` in any modern browser.
- **Drop into a site/repo:** copy `index.html` in (e.g. as your dashboard page,
  or as the repo's `index.html`) and deploy. It works as-is on GitHub Pages,
  Netlify, Cloudflare Pages, or any static host.

## What's inside

- **One file** — `index.html`, with all HTML, CSS, and JS inline.
- **Runs entirely in the browser.** No backend, nothing uploaded.
- **Loads with built-in demo data** so it looks populated out of the box. A user
  can drop in a CSV / OFX / QFX bank export to see their own numbers — parsing
  happens locally in the browser.
- **Fonts** (Outfit / Inter / JetBrains Mono) load from Google Fonts. With an
  internet connection you get the exact typography; offline it falls back to
  system fonts.

## Notes

- This is the **front-end / UI only** — no QuickBooks or automation wiring is
  included.
- Light and dark themes; toggle from the sidebar.
- Organization & education tool — **not** tax, accounting, investment, or legal
  advice.
